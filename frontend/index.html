        Tentang IF B SR

        IF B SR adalah portal kelas untuk mengelola informasi penting seperti pengumuman, jadwal kuliah, toko kelas, dan feedback anonim. Dibangun dengan front-end murni (HTML, CSS, JS) dan data tersimpan di database lokal.

        • Home: Pengumuman, Petugas Doa, Jadwal Hari Ini, Kata-kata, Feedback
        • Jadwal: Lihat jadwal per-hari, filter, dan cari
        • Toko: Katalog produk kelas dengan harga
        • Admin: Login untuk kelola konten
